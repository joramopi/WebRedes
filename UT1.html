Unidad 1: Introducción a las Redes de Computadoras

1.1. ¿Qué es una red de computadoras?

Imagina que tu computadora pudiera hablar con otras. Compartir archivos, jugar en equipo, imprimir en la misma impresora o simplemente acceder a Internet. Todo eso es posible gracias a una red de computadoras. En palabras simples, una red es un conjunto de dispositivos (como computadoras, impresoras o móviles) conectados entre sí para intercambiar información y recursos.

"Una red de computadoras es un sistema de interconexión entre dispositivos autónomos que se comunican para compartir información y recursos" (Lowe, 2016, p. 7).

1.2. Ventajas de usar redes

  • Compartir archivos y documentos sin usar USB.
  • Acceder a Internet desde varios dispositivos.
  • Imprimir desde varias computadoras a una misma impresora.
  • Hacer videollamadas o jugar en línea.
  • Hacer copias de seguridad automáticas.

"Las redes permiten compartir información, reducir costos y facilitar la comunicación" (Stallings, 2004, p. 14).

1.3. Elementos básicos de una red

Elemento Descripción Ejemplo
Dispositivos finales Computadoras, impresoras, teléfonos Laptop, celular
Medios de transmisión Por donde viajan los datos Cables, Wi-Fi
Dispositivos de red Ayudan a conectar y dirigir datos Switch, router
Software Programas que permiten la comunicación Sistema operativo, navegador

1.4. Clasificación de redes

LAN

(Local Area Network): Red pequeña, como la de tu casa o salón de clases.

Hasta 200m de alcance

MAN

(Metropolitan Area Network): Red que conecta oficinas o universidades dentro de una ciudad.

Hasta 50km de alcance

WAN

(Wide Area Network): Red extensa que conecta redes más pequeñas a nivel mundial.

Alcance mundial

"Las redes se clasifican según su tamaño y alcance geográfico" (Tanenbaum, 211, p. 23).

1.5. Tipos de conexión

Alámbrica

  • Usa cables
  • Más rápida y estable
  • Menos interferencias

Inalámbrica

  • Usa ondas de radio (Wi-Fi)
  • Más cómoda y portátil
  • Mayor flexibilidad de ubicación

1.6. Protocolos: el lenguaje de las redes

Para que los dispositivos se entiendan, deben hablar el mismo idioma. Ese "idioma" se llama protocolo. El más famoso es TCP/IP, que permite navegar por Internet.

"Los protocolos definen las reglas que rigen la comunicación entre dispositivos en una red" (Stallings, 2004, p. 29).

1.7. Modelo OSI: la torre de la comunicación

El modelo OSI (Open Systems Interconnection) es una forma de entender cómo viajan los datos de un dispositivo a otro. Tiene 7 capas, cada una con una función específica:

Modelo OSI - 7 Capas

7. Aplicación

Programas como navegador o correo

6. Presentación

Formato de los datos

5. Sesión

Gestión de la conexión

4. Transporte

TCP, control de flujo y errores

3. Red

Direcciones IP, enrutamiento

2. Enlace de datos

Direcciones MAC, detección de errores

1. Física

Cables, señales, conectores

"El modelo OSI es una referencia conceptual para entender la arquitectura de red" (Tanenbaum, 2011, p. 41).

1.8. Modelo TCP/IP: el modelo real

Es el modelo usado en Internet. Tiene 4 capas:

4

Aplicación

(equivale a las capas 5, 6 y 7 del OSI)

correo, web, etc.

3

Transporte

(equivale a la capa 4 del OSI)

usa TCP o UDP

2

Internet

(equivale a la capa 3 del OSI)

usa IP

1

Acceso a la red

(equivale a las capas 1 y 2 del OSI)

"El modelo TCP/IP es la base de la comunicación en Internet y redes modernas" (Lowe, 2016, p. 249).

1.9. Encapsulamiento de datos: el viaje de la información

Cuando envías un mensaje, este pasa por varias capas. Cada capa añade su propia "envoltura" al mensaje. Es como enviar un regalo: lo pones en una caja (capa 4), la metes en otra (capa 3), y así sucesivamente hasta llegar al sobre (capa 1). Al llegar al destino, se abre capa por capa.

Ejemplo de encapsulamiento:

4
"Hola mundo"
Datos
3
[Cabecera TCP] + "Hola mundo"
Segmento
2
[Cabecera IP] + [TCP] + "Hola mundo"
Paquete
1
[Cabecera Ethernet] + [IP] + [TCP] + "Hola mundo" + [Tráiler]
Trama

"El encapsulamiento permite que cada capa agregue información necesaria para el envío correcto de los datos" (Stallings, 2004, p. 36).

1.10. Ejemplo cotidiano: red en una cafetería

Una cafetería moderna puede tener:

  • Un router que brinda Wi-Fi a clientes.
  • Una impresora compartida por el personal.
  • Una caja registradora conectada a Internet.
  • Una computadora que administra pedidos y menús.

Todo eso forma una red de computadoras.
